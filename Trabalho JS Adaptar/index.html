Hardware

Aqui está a descrição de todos os componentes essenciais para o funcionamento do robô.

O Arduino é o cérebro do robô, conectando todos os outros componentes para que tudo fique funcional e programável. Sua programação pode ser customizada para qualquer uso como automações e outros projetos que envolvem eletrônica e programação.

O Módulo Bluetooth é conectado direto ao arduino para que mesmo receba entradas do usuário sem o uso de um cabo, mas sim um aplicativo no seu celular que conversa com o código dentro do arduino, para o robô realizar uma de suas funções essenciais, o movimento.

O Shield Motor é o que faz a comunicação entre os motores e o arduino, sem eles a programação do arduino não saberia como se comunicar com os motores.
Os Motores existem para exercer as funções motoras do robô, essas sendo, o movimento das armas e o movimento das rodas.

O Sensor de Fim de Curso é conectado ao arduino no intuito de comunicar ao código quando algo encosta no robô assim ativando as armas.
